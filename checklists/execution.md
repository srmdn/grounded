# Execution Checklist

- Scope of the current milestone is clear
- Main user and painful problem are still clear
- Constraints are understood
- The work is small enough to complete without drift
- The output format is defined
- Reversal or rollback is understood when relevant
- Basic checks are defined
- The work is usable in the real context it was built for
- Immediate review happens after execution
- The next bottleneck is captured before starting new work

## Software-Specific Additions

If the work includes a software deployment, also check:

- Branch state understood
- Environment variables accounted for
- Migrations understood
- Build or tests pass
- App starts successfully
- Basic smoke test passes
